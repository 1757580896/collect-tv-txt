# collect-tv-txt

自动收集整理大佬们的直播源，程序小白自用，如有侵权请联系。

| 文件名  | 更新频率                                       | 备注   |
|-------|------------------------------------------------|------------|
| （merged_output.txt） |  每日自动更新 | -   |
| （merged_output.m3u） |  每日自动更新 | -   |
| （blacklist_auto.txt） |  不定时更新 | -   |
